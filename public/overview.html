TikTokDictionary

This is a custom dictionary platform built for Gen-Z slang. It allows users to post words, vote on definitions, and see what is trending right now.

Goal: To create a premium, clean version of Urban Dictionary tailored for TikTok culture.

1. User Journey

Here is how a normal user uses the website:

  1. Visitor: A guest lands on the homepage and can search for words or see the "Word of the Day".
  2. Sign Up: To contribute, they go to the Register page and create an account.
  3. Submission: The user clicks "Submit Word" and adds a new slang term (e.g., "Rizz").
  4. Voting: Other users see this word and can upvote (Agree) or downvote (Disagree).
  5. Trending: If a word gets enough activity, it shows up on the "Trending" list on the homepage.

2. Project Modules Covered

I have built the following core features:

  • Authentication Module: secure Login and Registration pages.
  • Dictionary Module: capability to Add words, Define them, and add Examples.
  • Voting System: dynamic Up/Down voting logic that updates in real-time.
  • Search & Browse: filters to find words by Category (Gaming, TikTok, Slang, etc.).
  • Vertical Feed: a TikTok-style scrolling feed to explore random words.
  • Admin Panel: a dashboard to manage users and delete bad words.

3. How to Use

  • To Test Login: Go to /login and use username "admin" (or create a new one).
  • To Test Submitting: Click the "+" button or "Submit Word" in the menu.
  • To See Code: The main logic is in the app/Livewire folder.
